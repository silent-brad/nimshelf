* NimShelf

A minimal, self-hosted digital library for managing EPUB books. Built with Nim for performance and simplicity.

** Stack

- [[https://nim-lang.org][Nim]] — Systems programming language
- [[https://github.com/guzba/mummy][Mummy]] — Multi-threaded HTTP server
- [[https://github.com/enthus1ast/nimja][Nimja]] — Compiled template engine (Jinja2-like)
- [[https://github.com/treeform/debby][Debby]] — SQLite ORM with connection pooling
- [[https://picocss.com][Pico.css]] — Minimal CSS framework (CDN)
- [[https://htmx.org][HTMX]] — Dynamic UI without JavaScript
- [[https://libsodium.org][libsodium]] — Password hashing (Argon2) and session encryption
- [[https://github.com/guzba/zippy][Zippy]] — Compression (used by Mummy)
- [[https://nixos.org][Nix]] — Reproducible builds

** Features

- Upload and manage EPUB books
- Search by title or author
- Download books
- Session-based authentication with encrypted cookies
- HTMX-powered dynamic updates (no page reloads for upload/search/delete)

** Getting Started

*** Prerequisites

- Nim >= 2.0
- SQLite3 (~libsqlite3~)
- libsodium

*** With Nix

#+begin_src sh
nix develop
nimble build
./src/main
#+end_src

*** Without Nix

#+begin_src sh
nimble install
nimble build
./src/main
#+end_src
